What is the Internet?

The Internet is a worldwide network that connects computers, mobile phones, servers, and other digital devices. It is important because it allows people to share information, communicate quickly, and access learning materials from almost anywhere. Students use the Internet to research lessons, submit schoolwork, and watch educational videos. Families and friends use it for email, messaging, video calls, and social networking. Businesses also use the Internet for online shopping, banking, advertising, and customer service. Because of these uses, the Internet has become an important part of education, communication, and daily life.

My Learning Reflection

Website planning is important before creating a webpage because it helps the developer understand the goal of the website. It also helps identify the audience, the information they need, and the best way to organize the content. Planning makes it easier to decide what pages, links, images, and sections should be included. A good plan can save time because the developer already has a guide before writing the HTML code. It also helps make the website easier to use because the navigation and layout are prepared in advance. Planning supports accessibility because the designer can think about users with different needs before the page is finished. Overall, website planning helps create a clearer, more organized, and more useful webpage.
